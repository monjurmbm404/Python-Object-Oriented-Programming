# 🐍 Python OOP – Day 10: Advanced Operator Overloading

---

## 📌 Overview

Welcome to **Day 10 of Python OOP** 🚀

Today you will go deeper into **Operator Overloading**, where you learn how to customize how objects behave with operators like:

✔ `+  -  *  /`  
✔ `>  <  ==`  
✔ Built-in functions like `len()`

---

## 💡 What is Operator Overloading?

Operator Overloading means:

👉 Changing the default behavior of operators for custom classes

### 🧠 Simple Idea:

Instead of only using operators with numbers:

✔ You can use them with objects too

---

## 🎯 Why It is Important?

✔ Makes objects behave like real-world entities  
✔ Improves readability of code  
✔ Helps build real applications (banking, ecommerce, cart systems)  
✔ Enables mathematical modeling with objects

# ⚖ 01. Custom Comparisons

## 📁 01_custom_comparisons.py

### 📘 Explanation:

You can define how objects are compared:

👉 `>` `<` `==`

### 🧠 Example:

```python id="op1"
p1 > p2
p1 < p2
p1 == p2
```

### 🔑 Key Idea:

✔ Comparison is based on custom logic (like price, marks, etc.)
